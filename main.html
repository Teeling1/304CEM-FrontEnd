Quiz Management

On this site you can manage your quizzes you have created, aswell as quizzes you have completed.

For the purpose of this project you will be able to sign up and log into the quiz portal

This will then allow you access to the rest of the site, where quiz questions can be created, updated and deleted. Users can also be similalry managed, all in accordance with the marking scheme, using jquery, ajax, js, angular.js etc. where appropriate.

The API is built in Express Node.js using mongoose and mongoDB, and provides more than the required number of CRUD requests on a RESTful API.
